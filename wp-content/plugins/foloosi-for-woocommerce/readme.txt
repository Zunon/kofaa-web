Foloosi Payments plugin to accept credit card and debit payments.

== Description ==

The Foloosi Payments plugin allows you to accept credit card and debit payments via Foloosi payment gateway on your WordPress site easily.
This plugin will add a Foloosi Payment option on WooCommerce checkout page, when user choses Foloosi as Payment Method, he will have redirected to Foloosi popup to complete his transaction Foloosi will send that user back to your website along with transactions details.

One click payment via Foloosi with a remember me feature. Responsive design so it is compatible with all devices and browsers.


== Installation ==

In the WordPress administration panel:

1. Go to WooCommerce -> Settings section->Payments

2. You can see Foloosi. Click manage button

3. Add Foloosi merchant key and Save (You can take this merchant key from Foloosi merchant panel under API Setting tab

If you have any questions or issues, feel free to contact our technical support: info@foloosi.com

== Dependencies == 

1. Wordpress v3.9.2 and later
2. Woocommerce v2.4 and later
3. PHP v5.6.0 and later
4. php-curl extension


== Frequently Asked Questions ==
= Does this require an SSL certificate? =

Yes! In live mode, an SSL certificate must be installed on your site to use Foloosi Payment Gateway.

= Does this will support multi-currency support? =

Yes. Please get multi-currency enabled for your account. Once you have it enabled, you can install this plugin
